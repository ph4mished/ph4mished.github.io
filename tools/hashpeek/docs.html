Usage Guide

Basic Usage

./hashpeek [OPTIONS] [input]

Options Reference

Input Options

  • -t, --text <hashstring> - Analyze a single hash
  • -f, --file <path> - Analyze file

Extraction Options

  • -fd, --field '{N} X' - Capture segment at index N after splitting by delimiter X
  • -pb, --probe <lengths> - Probe for embedded hashes
  • -e-ctext, --extract-context <words> - Extract near context words
